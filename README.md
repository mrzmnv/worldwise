# WorldWise

A travel tracker app. Log cities you've visited by clicking on an interactive map, add notes and dates, and browse your visited countries.

Built while learning React — based on a course project.

## What I practiced
- React Router v6 (nested routes, programmatic navigation)
- Context API for global state
- Custom hooks
- Leaflet map integration
- Fake authentication flow
- URL as state (lat/lng in query params)

## Tech
- React 18
- React Router v6
- Context API
- Leaflet / React Leaflet
- Vite

## Run locally

```bash
npm install
npm run dev
```
